#

실제로 사용되고 있는 웹사이트 템플릿을 참고해 만들어보았습니다.

1. 해외에서 많이 쓰이는 One-Page-Scroll 형식의 웹사이트에
2. 이미지 슬라이더, 숫자 카운터, 이미지 갤러리, 움직이는 Bar 등 다양한 기능을 갖춘
3. 반응형 웹사이트라서
다양한 HTML/CSS/jQuery 기능을 연습할 수 있었습니다.

모든 코딩은 기존 웹사이트 코드를 보지 않고 제 방식대로 처음부터 만들었습니다.

#

어떤 브라우저 사이즈도 감당할 수 있습니다.

단순히 사이즈만 줄이는 것이 아니라 태블릿/스마트폰에 맞춰 레이아웃도 변경합니다. 윈도우창 사이즈를 조절하며 구경해보세요!
1200px, 992px, 768px, 576px을 기준으로 반응합니다. 작은 화면에서는 태블릿/스마트폰용 메뉴로 바뀝니다.

#

숫자도 퍼센테이지 표시도 가만히 있으면 재미없잖아요.

jQuery를 이용해 숫자와 퍼센테이지가 점점 움직이도록 만들었습니다. 스크롤 함수를 통해 화면에 보이자마자 움직입니다.

#

간단한 이미지 갤러리는 제가 직접 코딩할 수 있습니다.

사진 분류도 가능하고, 윈도우 사이즈에 맞춰 행/열도 자동으로 바뀝니다.

#

마우스를 갖다대고 싶게 만드는 웹 애니메이션을 소개합니다.

Hover 기능을 이용해 사용자들이 심심하지 않도록 동적인 웹사이트를 구현했습니다.
