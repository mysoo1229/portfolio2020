#

왜 하필 인터파크를 연습해보았냐면요,

1. 한페이지에 다양한 메뉴가 있고
2. jQuery의 여러 기능을 적용할 수 있고
3. 실제 사이트의 header와 footer 등을 연습하기에 적합하다고 생각했습니다.

모든 코딩은 실제 웹사이트 코드를 보지 않고 제 방식대로 처음부터 만들었습니다.

#

메뉴 속 메뉴! 2-Depth 메뉴를 4번 이상 사용했습니다.

ul 안에 ul을 넣기도 하고

div카드 안에 3개의 div를 넣어 화살표를 누르면 변환되게 했습니다.
이 외에도 사이트 곳곳에 여러가지 메뉴가 많아요.

#

사용자에게 정보를 떠먹여주며 시선을 빼앗아보겠습니다.

setInterval 함수를 이용해 자동으로 새로운 배너 & 새로운 메뉴를 보여드립니다.

#
